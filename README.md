#  Weight & Volume Calculator (Python)

##  Description
This project is a beginner-friendly Python program that converts weight between kilograms and pounds and calculates the volume of an object using user input.

It helps practice:
- User input and output
- if / elif / else conditions
- Basic math operations
- Clean Python structure

##  Features
- Convert kilograms to pounds
- Convert pounds to kilograms
- Calculate volume using length × width × height
